# ipl-project
“IPL Analysis Dashboard with Player Stats, Points Table, Orange Cap and Purple Cap”


**IPL Power BI Project**

 Dashboard Overview

This project includes:
Player-wise Total Runs and Wickets
Orange Cap (Highest Runs Player)
Purple Cap (Highest Wickets Player)
Team-wise Points Table

Key Insights

Orange Cap → Player with highest total runs
Purple Cap → Player(s) with highest wickets (handles ties)
Points Table → Based on wins (2 points per win)
